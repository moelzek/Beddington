# Beddington soothe assets

These are synthetic local placeholder sounds for Tier 1 testing:

- `white_noise.wav` — low-pass filtered noise.
- `uterine_whoosh.wav` — stronger synthetic womb-like rumble and whoosh.
- `heartbeat.wav` — heartbeat-style pulses, not a medical recording.
- `soothing_music.wav` — simple generated tones.

They are quiet mono 16 kHz WAV files generated by
[`scripts/generate_soothe_assets.py`](../../scripts/generate_soothe_assets.py).
The application can loop the selected preset for longer configured play windows.
Parents should choose one mode rather than cycle through all sounds automatically.
Keep first playback tests at low volume on a laptop before trying the Pi speaker
or amplifier. These are output-test assets, not evidence that any sound will
settle a baby.
